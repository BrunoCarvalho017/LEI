#App
[Estado] :	-Consegue executar o analisador.py;
			-É possível verificar a criação do resultado.xslx;

[Fazer] : 	-A criação do objeto JSON com a informação retirada do analisador, 
			que será usada para a criação da página;
			-Criação da página de submissão dos metadados; 

[ATENÇÃO] : 	-Executar o servidor com _npm start_ em vez de _nodemon_;
				-Se o ficheiro .gitignore estiver a funcionar, é suposto não terem o 
				node_modules atualizado, por esse motivo necessitam de fazer _npm i_;
				-Quando mexerem na App e precisarem de instalar algo, não se esqueçam
				de o fazer com a tag --save
